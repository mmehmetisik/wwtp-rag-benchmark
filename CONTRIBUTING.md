# Contributing to RAG Under Pressure Benchmark

Thank you for your interest in this project!

## How to Contribute

### 🐛 Reporting Issues
- Use the GitHub Issues tab
- Describe the problem clearly with steps to reproduce
- Include relevant data or logs if applicable

### 💡 Suggesting Improvements
- Open an issue with the `enhancement` label
- Describe the proposed change and its motivation

### 🔬 Adding New Model Results
If you have run the benchmark engine against a model not included in this study:
1. Fork the repository
2. Add your results to `data/` following the existing format
3. Open a Pull Request with a description of the model and methodology

### 📋 General Guidelines
- Keep discussions constructive and on-topic
- Provide evidence for any factual claims about model performance
- Respect the benchmark methodology when comparing results
